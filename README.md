# 감성 일기장

  ## 사용한 라이브러리
  * React
  * React-router-dom
    
     
 ## 주요기능
 
     * CRUD 
     * 날자와 최신순 정렬

## 신경쓴부분
* 라이브러리를 최대한 사용하지않고 만들어보려고 react Hooks 를 이용하였습니다.
* 상태관리는 useState 와 useReducer useContext 를 이용하고,
* 최적화를 해보기위해 useCallback useMemo 를 사용했습니다.
